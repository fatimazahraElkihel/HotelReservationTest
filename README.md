# RoomReservationTest

Simple Java application implementing a hotel reservation system.

## Description
The system allows:
- Creating users with a balance
- Creating hotel rooms
- Booking rooms for a specific period
- Preventing double bookings
- Updating room information without impacting previous bookings

## Technical Choices
- Java only
- No database
- No repositories
- Uses ArrayList
- Booking stores room and user data at booking time

## How to Run
Run the `Main` class to execute the test scenario and display results.
